CSS Transition

The transition property transition an element from state to state. For example like when you have a hover event. It makes the transition happen over the specified time.

CSS Transformation

The transform property will take an element and transform it in some way. There are few different things we could do

  • Scale - Scales an element by a factor
  • Rotate - Rotate the element using degrees
  • Skew - Skew an element using deg
  • Translate - Moves an element on the X and Y axis
  • Prespective - Will give a child element 3D prespective when rotated. A lower number will mean a more extreme prespective.
  • Marix - Matrix is simply using multiple of these at the same time but a true matrix is convert to numbers.

    • Please Note - Transform can be used with transition and/ or animation but don't have to be. If used on its own, it will just be static.
